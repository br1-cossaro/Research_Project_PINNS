Solving the 3D Linear Elasticity Equation with DeepXDE¶

Problem description¶

The behavior of an elastic material under external forces is governed by the linear elasticity equations. These equations describe how the material deforms (displacements) in response to applied forces, based on its mechanical properties.

In the absence of body forces and assuming isotropic and homogeneous material properties, the equilibrium equations in 3D simplify to the Poisson equation for each component of the displacement vector $\mathbf{u} = (u, v, w)$:

$$ -\nabla^2 u = f_x(\mathbf{x}) $$

$$ -\nabla^2 v = f_y(\mathbf{x}) $$

$$ -\nabla^2 w = f_z(\mathbf{x}) $$

Where:

  • $\nabla^2$ is the Laplacian operator, representing the sum of second derivatives with respect to spatial variables.
  • $u, v, w$ are the displacement components in the $x$, $y$, and $z$ directions, respectively.
  • $f_x, f_y, f_z$ are the components of the external force per unit volume.
  • $\mathbf{x} = (x, y, z)$ represents the spatial coordinates within the domain.

Problem setup¶

We consider a cubic domain $\Omega = [0, 1]^3$ with zero displacement on all boundaries (Dirichlet boundary conditions). This means the cube is fixed at its boundaries, and any deformation occurs within the interior due to the applied forces.

Exact solution¶

To validate our numerical solution, we choose an exact analytical solution for the displacement components:

$$ u(\mathbf{x}) = \sin(\pi x) \sin(\pi y) \sin(\pi z) $$

$$ v(\mathbf{x}) = \sin(\pi x) \sin(\pi y) \sin(\pi z) $$

$$ w(\mathbf{x}) = \sin(\pi x) \sin(\pi y) \sin(\pi z) $$

From this exact solution, we can derive the corresponding source terms $f_x, f_y, f_z$ using the Laplacian:

$$ -\nabla^2 u = -\nabla^2 v = -\nabla^2 w = 3\pi^2 \sin(\pi x) \sin(\pi y) \sin(\pi z) $$

Thus, the source terms are:

$$ f_x(\mathbf{x}) = f_y(\mathbf{x}) = f_z(\mathbf{x}) = 3\pi^2 \sin(\pi x) \sin(\pi y) \sin(\pi z) $$

Implementation in DeepXDE¶

To solve this problem using DeepXDE, will be needed to define the geometry, governing equations, boundary conditions, and the neural network architecture.

Geometry definition¶

We define the cubic domain $\Omega$ using DeepXDE's geometry module:

import deepxde as dde

# Define the cubic domain
geom = dde.geometry.Cuboid([0, 0, 0], [1, 1, 1])

Governing Equations¶

For each displacement component, we define a separate partial differential equation (PDE) corresponding to the Poisson equation:

For $u$:

$$ -\left( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} \right) = f_x(\mathbf{x}) $$

In DeepXDE, we implement this as:

def pde_u(x, u):
    u_xx = dde.grad.hessian(u, x, i=0, j=0)
    u_yy = dde.grad.hessian(u, x, i=1, j=1)
    u_zz = dde.grad.hessian(u, x, i=2, j=2)
    return -(u_xx + u_yy + u_zz) - f_component(x)

Source Term Function¶

The source term $f_x(\mathbf{x})$ (similarly for $f_y$ and $f_z$) is defined using PyTorch functions to ensure compatibility with automatic differentiation of Deepxde, that facilitates the work of the library.

def f_component(x):
    return 3 * torch.pi**2 * torch.sin(torch.pi * x[:, 0:1]) * \
           torch.sin(torch.pi * x[:, 1:2]) * torch.sin(torch.pi * x[:, 2:3])

Boundary Conditions¶

We apply Dirichlet boundary conditions, setting the displacement to zero on all boundaries:

def boundary(x, on_boundary):
    return on_boundary

bc_u = dde.DirichletBC(geom, lambda x: 0, boundary)
bc_v = dde.DirichletBC(geom, lambda x: 0, boundary)
bc_w = dde.DirichletBC(geom, lambda x: 0, boundary)

Neural Network Architecture¶

We use a feedforward neural network with the following specifications:

  • Input dimension: 3 (for $x$, $y$, $z$)
  • Output dimension: 1 (displacement component)
  • Hidden layers: 4 layers with 50 neurons each
  • Activation function: Hyperbolic tangent ($\tanh$)
  • Initializer: Glorot normal

The configuration is defined as follows:

layers = [3] + [50] * 4 + [1]
activation = "tanh"
initializer = "Glorot normal"

net_u = dde.maps.FNN(layers, activation, initializer)
net_v = dde.maps.FNN(layers, activation, initializer)
net_w = dde.maps.FNN(layers, activation, initializer)

Model Compilation and Training¶

We create a Model object for each displacement component, compile it with the Adam optimizer, and train it:

model_u = dde.Model(data_u, net_u)
model_u.compile("adam", lr=1e-3)
losshistory_u, train_state_u = model_u.train(iterations=50000)

# Repeat for model_v and model_w

Data Definition¶

The PDE object combines the geometry, the PDE, and the boundary conditions:

data_u = dde.data.PDE(
    geom,
    pde_u,
    bc_u,
    num_domain=10000,

# Similarly for data_v and data_w

Evaluation and Visualization of the Solution¶

After training, we evaluate the predictions of the neural network and compare them with the exact solution.

Model Evaluation¶

We generate a set of points within the domain and compute the predicted and exact displacements:

X = geom.uniform_points(1000, True)
u_pred = model_u.predict(X)
u_exact = exact_solution(X)

Relative $L^2$ Error Calculation¶

To quantify the accuracy of the predictions, we compute the relative $L^2$ error:

error_l2_u = dde.metrics.l2_relative_error(u_exact, u_pred)

def exact_solution(x):
    return np.sin(np.pi * x[:, 0:1]) * np.sin(np.pi * x[:, 1:2]) * np.sin(np.pi * x[:, 2:3])

# Define the components of the source function f_x, f_y, f_z

def f_component(x):
    return 3 * torch.pi**2 * torch.sin(torch.pi * x[:, 0:1]) * torch.sin(torch.pi * x[:, 1:2]) * torch.sin(torch.pi * x[:, 2:3])

# Define the PDE for each component
def pde_u(x, u):
    u_xx = dde.grad.hessian(u, x, i=0, j=0)
    u_yy = dde.grad.hessian(u, x, i=1, j=1)
    u_zz = dde.grad.hessian(u, x, i=2, j=2)
    return -(u_xx + u_yy + u_zz) - f_component(x)

def pde_v(x, v):
    v_xx = dde.grad.hessian(v, x, i=0, j=0)
    v_yy = dde.grad.hessian(v, x, i=1, j=1)
    v_zz = dde.grad.hessian(v, x, i=2, j=2)
    return -(v_xx + v_yy + v_zz) - f_component(x)

def pde_w(x, w):
    w_xx = dde.grad.hessian(w, x, i=0, j=0)
    w_yy = dde.grad.hessian(w, x, i=1, j=1)
    w_zz = dde.grad.hessian(w, x, i=2, j=2)
    return -(w_xx + w_yy + w_zz) - f_component(x)

Visualization¶

The visualization of the predicted displacement field within the unit cube was conducted through two distinct figures, each providing a unique perspective on the deformation behavior.

Surface Plot at ( z = 0.5 )¶

The first figure focuses on a cross-sectional analysis of the displacement component ( u ) at the mid-plane of the cube, specifically at ( z = 0.5 ). To achieve this, points within a narrow tolerance range around ( z = 0.5 ) were selected from the uniformly distributed dataset. This selection ensures that the analysis is concentrated on the central slice of the geometry, providing a clear view of the displacement variations in the ( x ) and ( y ) directions.

Complete 3D Deformation Visualization¶

The second figure offers a comprehensive view of the entire displacement field within the unit cube, encapsulating the collective deformation across all three spatial dimensions. This visualization integrates both the original and deformed meshes to illustrate the extent and direction of displacements.

import numpy as np
import plotly.graph_objs as go
from scipy.interpolate import griddata

# Number of points per side of the cube
n = 20

# Function to generate surface points of the cube

def generate_cube_surface_points(n):

    lin = np.linspace(0, 1, n)
    
    # Create a meshgrid from the linearly spaced array for X and Y coordinates
    X, Y = np.meshgrid(lin, lin)
    
    # Flatten the meshgrid arrays to make them one-dimensional
    X = X.flatten()
    Y = Y.flatten()
    

    faces = []
    
    # Face where x = 0 (left face of the cube)
    faces.append(np.vstack([np.zeros_like(X), Y, X]).T)
    
    # Face where x = 1 (right face of the cube)
    faces.append(np.vstack([np.ones_like(X), Y, X]).T)
    
    # Face where y = 0 (front face of the cube)
    faces.append(np.vstack([X, np.zeros_like(Y), Y]).T)
    
    # Face where y = 1 (back face of the cube)
    faces.append(np.vstack([X, np.ones_like(Y), Y]).T)
    
    # Face where z = 0 (bottom face of the cube)
    faces.append(np.vstack([X, Y, np.zeros_like(X)]).T)
    
    # Face where z = 1 (top face of the cube)
    faces.append(np.vstack([X, Y, np.ones_like(X)]).T)
    
    # Combine all the faces into a single array of surface points
    surface_points = np.concatenate(faces, axis=0)
    
    return surface_points

# Generate surface points of the cube
surface_points = generate_cube_surface_points(n)

# Predict displacements using the trained models

u_pred_surface = model_u.predict(surface_points)
v_pred_surface = model_v.predict(surface_points)
w_pred_surface = model_w.predict(surface_points)

# Add the predicted displacements to the original surface points
deformed_points = surface_points + np.hstack([u_pred_surface, v_pred_surface, w_pred_surface])
